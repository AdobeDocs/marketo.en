# Upgrade the Marketo Solution for Microsoft Dynamics {#upgrade-the-marketo-solution-for-microsoft-dynamics}

Upgrade the Marketo Solution for Microsoft Dynamics - Marketo Docs - Product Documentation

When a new Microsoft Dynamics solutions is released, you can download the upgrade from the Admin area in your account.

>[!NOTE]
>
>**Admin Permissions Required**

>[!CAUTION]
>
>It is imperative you download the latest Marketo Solution *before* performing any upgrade.

##### 1. Go to the Admin area. {#upgradethemarketosolutionformicrosoftdynamics-gototheadminarea.}

![](assets/admin.png)

##### 2. Click Microsoft Dynamics. {#upgradethemarketosolutionformicrosoftdynamics-clickmicrosoftdynamics.}

![](assets/image2015-3-16-10-3a51-3a25.png)

##### 3. Select Download Marketo Solution. {#upgradethemarketosolutionformicrosoftdynamics-selectdownloadmarketosolution.}

![](assets/image2015-3-16-10-3a52-3a1.png)

##### 4. Select the appropriate solution for your Microsoft Dynamics version. {#upgradethemarketosolutionformicrosoftdynamics-selecttheappropriatesolutionforyourmicrosoftdynamicsversion.}

![](assets/msd-online.png)

Awesome! A zip file of the solution will now be downloaded to your device. If you are unfamiliar with the install steps, please contact your Dynamics Admin.
