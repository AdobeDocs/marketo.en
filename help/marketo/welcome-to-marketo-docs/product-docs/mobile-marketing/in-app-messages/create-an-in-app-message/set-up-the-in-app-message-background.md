# Set Up the In-App Message Background {#set-up-the-in-app-message-background}

Set Up the In-App Message Background - Marketo Docs - Product Documentation

Selecting a message background is an important step in finishing up your in-app message.

>[!TIP]
>
>For background images, textures and gradients typically work best.

##### 1. In the editor, click the background area. {#setupthein-appmessagebackground-intheeditor-clickthebackgroundarea.}

![](assets/image2016-5-9-8-3a38-3a1.png)

##### 2. To set a color, click Color, then the color selection square. Select the color by clicking it or entering the Hex or RGB codes on the color picker. {#setupthein-appmessagebackground-tosetacolor-clickcolor-thenthecolorselectionsquare.selectthecolorbyclickingitorenteringthehexorrgbcodesonthecolorpicker.}

![](assets/image2016-5-9-8-3a46-3a59.png)

>[!NOTE]
>
>Plain white, the default, is usually a winner.

##### 3. If you want to use a background, click Image and then the Select Image link. {#setupthein-appmessagebackground-ifyouwanttouseabackground-clickimageandthentheselectimagelink.}

![](assets/image2016-5-9-8-3a52-3a43.png)

##### 4. Choose from files uploaded to the the Design Studio. Click Select. {#setupthein-appmessagebackground-choosefromfilesuploadedtothethedesignstudio.clickselect.}

![](assets/image2016-5-9-9-3a0-3a2.png)

>[!NOTE]
>
>Image size is limited to 1440 x 2560 px and 5MB.

##### 5. Evaluate the results. Oops–maybe a plain color, texture, or gradient will work better! {#setupthein-appmessagebackground-evaluatetheresults.oops–maybeaplaincolor-texture-orgradientwillworkbetter!}

![](assets/image2016-5-9-9-3a2-3a33.png)

##### 6. Apply an optional image border. The default setting is Off. First, select the color by clicking it or entering the Hex or RGB numbers on the color picker. {#setupthein-appmessagebackground-applyanoptionalimageborder.thedefaultsettingisoff.first-selectthecolorbyclickingitorenteringthehexorrgbnumbersonthecolorpicker.}

![](assets/image2016-5-9-9-3a54-3a8.png)

##### 7. Click the arrows to vary the border width. This 12-pixel-wide green border changes the look of the in-app message. {#setupthein-appmessagebackground-clickthearrowstovarytheborderwidth.this12-pixel-widegreenborderchangesthelookofthein-appmessage.}

![](assets/image2016-5-9-9-3a58-3a38.png)

##### 8. Select a corner radius using the slider. Select a position from left to right: 0, 4, 8, 12, or 16 pixels. The 8-pixel (center) selection is the default. {#setupthein-appmessagebackground-selectacornerradiususingtheslider.selectapositionfromlefttoright-0-4-8-12-or16pixels.the8-pixel(center)selectionisthedefault.}

![](assets/image2016-5-6-9-3a39-3a28.png)

##### 9. Check the box to set the background tap action (off by default). {#setupthein-appmessagebackground-checktheboxtosetthebackgroundtapaction(offbydefault).}

![](assets/image2016-5-9-10-3a6-3a10.png)

>[!NOTE]
>
>For tap actions for backgrounds, you can set up different actions for Apple and Android platforms. For example, deep links are handled differently for Apple and Android. If your message is going only to one platform or the other, leave the other one in the default setting or select **None**.

To finish the job, your last step is to [set up the Dismiss button](set-up-the-dismiss-button-and-approve-the-message.md).
