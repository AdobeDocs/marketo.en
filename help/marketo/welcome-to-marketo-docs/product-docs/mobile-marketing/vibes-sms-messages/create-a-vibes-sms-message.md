# Create a Vibes SMS Message {#create-a-vibes-sms-message}

Create a Vibes SMS Message - Marketo Docs - Product Documentation

Here's how to create a Vibes SMS message.

##### 1. Go to Marketing Activities and right-click on a program. {#createavibessmsmessage-gotomarketingactivitiesandright-clickonaprogram.}

![](assets/mobile-right-click-hand.jpg)

##### 2. Click New Local Asset. {#createavibessmsmessage-clicknewlocalasset.}

![](assets/new-local-asset-hand.jpg)

>[!TIP]
>
>Alternatively, you can click the **New** drop-down.

##### 3. Click SMS Message. {#createavibessmsmessage-clicksmsmessage.}

![](assets/new-local-asset-selection-hand.jpg)

##### 4. Enter a name and an optional description for the new SMS message and click Create. {#createavibessmsmessage-enteranameandanoptionaldescriptionforthenewsmsmessageandclickcreate.}

![](assets/new-sms-message-offer-ends-soon-hands.jpg)

##### 5. Click Edit Draft. {#createavibessmsmessage-clickeditdraft.}

![](assets/edit-draft-hand.jpg)

##### 6. In the message editor, click inside the blue bubble and begin entering text. {#createavibessmsmessage-inthemessageeditor-clickinsidethebluebubbleandbeginenteringtext.}

![](assets/message-text-pencil.jpg)

>[!NOTE]
>
>US and Canada limits are different, at 160 and 130 characters respectively. If you exceed these character limits, your message could be split. While we show when you exceed the Canadian limit, the editor is optimized for US and splits the message based on the US limit.

##### 7. Click Token in the Insert menu to add a token to your message. {#createavibessmsmessage-clicktokenintheinsertmenutoaddatokentoyourmessage.}

![](assets/add-token-real-hand.jpg)

>[!NOTE]
>
>Adding a token may cause the message to exceed the character limit. The message then will split, resulting in additional charges.

##### 8. Click Link in the Insert menu to add a link to your message. {#createavibessmsmessage-clicklinkintheinsertmenutoaddalinktoyourmessage.}

![](assets/full-message-link-hand.jpg)

##### 9. Select a link type. Marketo Landing Page is the default. If you go with that, you need to select the landing page from the drop-down and click Insert. {#createavibessmsmessage-selectalinktype.marketolandingpageisthedefault.ifyougowiththat-youneedtoselectthelandingpagefromthedrop-downandclickinsert.}

![](assets/insert-link-real-hands.jpg)

>[!NOTE]
>
>The two tracking links are selected by default.

##### 10. If you want to use an external URL instead, click the External URL button and enter the URL in the URL field. Click Insert. {#createavibessmsmessage-ifyouwanttouseanexternalurlinstead-clicktheexternalurlbuttonandentertheurlintheurlfield.clickinsert.}

![](assets/insert-link-url-hands.jpg)

##### 11. The link displays in the message. {#createavibessmsmessage-thelinkdisplaysinthemessage.}

![](assets/link-added.jpg)

>[!NOTE]
>
>Marketo displays a link preview of the branded tracking domain. If you clear the mkt_tok link checkbox, the link is changed. Clear the Track Link checkbox too and the URL will be shortened to its basic length (for example, www.mygooglepage.com).

If you insert more than the US limit, the editor breaks your message into sections. There is an absolute total limit of 900 characters. After reaching that limit, the message will be automatically truncated when it's sent to its audience.   ![](assets/image2016-7-27-16-3a20-3a16.png)

>[!NOTE]
>
>Character count reflects only the characters contained in the lowest message.
