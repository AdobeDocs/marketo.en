# Add Subdomains in Account Settings {#add-subdomains-in-account-settings}

Add Subdomains in Account Settings - Marketo Docs - Product Documentation

Here's how to add subdomains to your primary domains in Account Settings. This will allow you to manage subdomains related to the specific RTP Javascript of your primary domain. It is recommended to deploy the Javascript tag to any added subdomains.

##### 1. In Web Personalization, go to Account Settings. {#addsubdomainsinaccountsettings-inwebpersonalization-gotoaccountsettings.}

![](assets/image2014-12-1-23-3-12.png)

##### 2. On the Domain Configuration page, you'll see a list of all the primary domains associated with your account. Each section lists the primary domain first (highlighted below), followed by any subdomains. Click Add Subdomain. {#addsubdomainsinaccountsettings-onthedomainconfigurationpage-you'llseealistofalltheprimarydomainsassociatedwithyouraccount.eachsectionliststheprimarydomainfirst(highlightedbelow)-followedbyanysubdomains.clickaddsubdomain.}

![](assets/highlightprimary2.png)

##### 3. Click Add. {#addsubdomainsinaccountsettings-clickadd.}

![](assets/add.png)

##### 4. Enter the subdomain URL. Click Add or Delete to manage your list of subdomains, then click OK when you're done. {#addsubdomainsinaccountsettings-enterthesubdomainurl.clickaddordeletetomanageyourlistofsubdomains-thenclickokwhenyou'redone.}

![](assets/newsubdomain.png)

##### 5. Your newly added subdomain will now be listed. {#addsubdomainsinaccountsettings-yournewlyaddedsubdomainwillnowbelisted.}

![](assets/finalnew.png)

>[!NOTE]
>
>If you want to add *primary* domains to your account, please contact [Marketo Support](http://docs.marketo.com/cdn-cgi/l/email-protection#0c7f797c7c637e784c616d7e67697863226f6361).
