# Understanding Event Programs {#understanding-event-programs}

Understanding Event Programs - Marketo Docs - Product Documentation

>[!NOTE]
>
>**Availability**
>
>Not all customers have purchased this functionality. Contact your sales rep for details.

#### Overview {#understandingeventprograms-overview}

An event program&nbsp;is a type of program that can be used to automate&nbsp;your offline or online events. Think of an event program as anything that has attendees at a specific date and time.&nbsp;

>[!NOTE]
>
>**Example**
>
>* Webinars
>* Parties
>* Tradeshows
>* Meetups
>

Events can automatically connect to your [webinar provider](launchpoint-event-partners.md)&nbsp;as well as sync with an [iPad check-in app](../../../../../welcome-to-marketo-docs/product-docs/core-marketo-concepts/mobile-apps/event-check-in/check-people-into-your-event-from-your-tablet.md).
