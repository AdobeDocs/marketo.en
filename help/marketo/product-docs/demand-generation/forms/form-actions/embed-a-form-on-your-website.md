# Embed a Form on Your Website {#embed-a-form-on-your-website}

Access a form's embed code to host it on your own website.

   >[!PREREQUISITES]
   >
   >The form must be approved for the embed code to be available.

1. Find and select the desired form.

   ![](assets/embed-a-form-on-your-website-1.png)

1. On the right of the form details screen, click **[!UICONTROL Embed code]**.

   ![](assets/embed-a-form-on-your-website-2.png)

   >[!CAUTION]
   >
   >**[Form Prefill](/help/marketo/product-docs/administration/settings/edit-landing-page-settings.md)** does not work when using the form embed code on your own pages _or_ a Marketo landing page. Form Prefill is only meant to work when the form is used in a Marketo landing page via the Insert Element option.

1. In the _Standard_ tab, click **[!UICONTROL Copy Text]**. Click **[!UICONTROL Close]** when done.

   ![](assets/embed-a-form-on-your-website-3.png)

   >[!NOTE]
   >
   >For Lightbox code, see [Use a Form in a Lightbox](/help/marketo/product-docs/demand-generation/forms/form-actions/use-a-form-in-a-lightbox.md).

1. Give the embed code to your web developer. 

   After the code is embedded on your website, any changes to the form in Marketo Engage will be pushed to your site upon form approval. You do not need to make changes to the code.

   >[!TIP]
   >
   >If your developer wants to customize the look or access advanced API functions, show them the [Forms 2.0 Developers Page](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/javascriptapi/forms-api-reference).
