# Remove from List {#remove-from-list}

#### Overview {#overview}

Since you have the ability to ** [Add to List](add-to-list.md)**, you also need a way to remove people from lists.

![](assets/image2014-9-22-10-3a44-3a3.png)

#### Usage {#usage}
