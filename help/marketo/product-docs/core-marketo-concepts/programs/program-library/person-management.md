## Conflict Rules {#conflict-rules}

* **Program Tags** 
   * Create tags in this subscription (Default) - _Recommended_  
   * Ignore

* **Landing Page template with the same name**
   * Copy original template (Default) - _Recommended_ 
   * Use destination template 

* **Images with the same name**
   * Keep both files (Default) - _Recommended_ 
   * Replace item in this subscription  

* **Email templates with the same name**
   * Keep both templates (Default) - _Recommended_ 
   * Replace existing template

## Best Practices {#best-practices}

* Consider adding additional Smart Campaigns to address each of the lifecycle status needs you may be tracking in your organization. Each campaign built in this program is meant to be an example of the best practice build and not specific to all use cases. Remember to update the Smart Campaigns to address your specific person lifecycle management process.  

* Consider updating the naming convention of this program example to align with yours.
