# Set the Default Currency {#set-default-currency}

Learn how to view and edit the default currency for your Marketo Engage subscription.

>[!NOTE]
>
>**Admin permissions required.**

1. Go to the **[!UICONTROL Admin]** area.

   ![](assets/set-default-currency-1.png)

1. Click **[!UICONTROL Location]**.

   ![](assets/set-default-currency-2.png)

1. Click **[!UICONTROL Edit]** in [!UICONTROL Subscription Currency Settings].

   ![](assets/set-default-currency-3.png)

1. Select the currency format of your choice and click **[!UICONTROL Save]**.

   ![](assets/set-default-currency-4.png)
