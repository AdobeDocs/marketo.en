# Program QA {#program-qa}

Audit your programs for best practices across all components, such as emails, landing pages, campaigns, and more.

There are two ways to access this Agent.  

Click on the agent from the center window click the drop down on the top right

click on the  Use this Agent button

and in the left pane of the prompt dialog box, the default text will appear.

Then select the Program that you would like to QA in the right pane

A summary of the program will appear in the center window providing an overview of Program and what was created for it.

Or, click on the Agent in the main Conversational AI screen. 

The AI Assistant will show the default text to show what Agent will be executed.

Then select the program on the right that needs to be QA'd.

and the same summary will appear in the center console
